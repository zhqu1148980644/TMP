写一个函数，求两个整数之和，要求在函数体内不得使用 “+”、“-”、“*”、“/” 四则运算符号。

 

```
示例:

输入: a = 1, b = 1
输出: 2
```
 

#### 提示：

-    a, b 均可能是负数或 0
-    结果不会溢出 32 位整数

#### Solutions


1. ##### straight forward

- `01 + 11 = 01 ^ 11 + ((01 & 11) << 1) = 10 + 10 = 100`

```c++
class Solution {
public:
    int add(int a, int b) {
        while (b) {
            int sum = a ^ b;
            b = (unsigned int)(a & b) << 1;
            a = sum;
        }

        return a;
    }
};
```
